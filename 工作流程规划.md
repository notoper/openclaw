# 侨和工作室 - 自动化工作流程规划

## 📅 每日工作流程

```
12:00 探子启动
    ↓
搜索当天新加坡相关新闻热点
    ↓
生成3个选题标题
    ↓
发送给用户确认
    ↓
用户选择1个标题
    ↓
探子整理相关素材
    ↓
    ├→ 薇薇（公众号）→ 撰写公众号文章
    └→ 红红（小红书）→ 撰写小红书笔记
    ↓
两者完成工作
    ↓
记录到飞书多维表格
    ↓
各自记录工作日志
```

---

## 🤖 各Agent职责

### 探子（Researcher）
**触发时间：** 每天中午12:00  
**职责：**
1. 搜索当天新加坡相关新闻热点
2. 生成3个选题标题
3. 发送给用户确认
4. 根据用户选择，整理相关素材
5. 将素材传递给薇薇和红红

**输出：**
- 3个选题标题
- 选定标题的相关素材包

---

### 薇薇（Weiwei）
**触发：** 收到探子的素材后  
**职责：**
1. 基于素材创作公众号文章
2. 进行公众号排版
3. 记录到飞书多维表格
4. 撰写工作日志

**输出：**
- 公众号文章（Markdown格式）
- 飞书表格记录
- 工作日志

---

### 红红（Honghong）
**触发：** 收到探子的素材后  
**职责：**
1. 基于素材创作小红书笔记
2. 进行小红书排版
3. 记录到飞书多维表格
4. 撰写工作日志

**输出：**
- 小红书笔记（emoji+hashtag格式）
- 飞书表格记录
- 工作日志

---

## 📊 飞书多维表格设计

### 表格名称：侨和工作室-内容生产记录表

### 字段设计：

| 字段名称 | 字段类型 | 说明 |
|---------|---------|------|
| 日期 | 日期 | 自动填写当天日期 |
| 选题标题 | 多行文本 | 用户选定的标题 |
| 素材来源 | 多选 | 探子提供的素材来源 |
| 公众号文章 | 附件 | 薇薇的文章 |
| 公众号状态 | 单选 | 待撰写/撰写中/已完成 |
| 小红书笔记 | 附件 | 红红的笔记 |
| 小红书状态 | 单选 | 待撰写/撰写中/已完成 |
| 完成时间 | 日期时间 | 实际完成时间 |
| 备注 | 多行文本 | 其他说明 |
