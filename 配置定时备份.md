# 配置定时备份指南

## ⏰ 方案1：使用系统cron（推荐）

### 步骤1：编辑crontab

```bash
crontab -e
```

### 步骤2：添加定时任务

在打开的编辑器中添加以下行：

```bash
# 每小时整点执行自动备份
0 * * * * cd /root/.openclaw/workspace-researcher && ./auto-backup.sh >> /var/log/workspace-backup.log 2>&1
```

### 步骤3：保存并退出

- 如果使用nano编辑器：按 `Ctrl+O` 保存，`Ctrl+X` 退出
- 如果使用vim编辑器：按 `Esc`，输入 `:wq` 保存并退出

## 📝 cron表达式说明

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期 (0-7，0和7都是周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

**我们的配置：** `0 * * * *` = 每小时整点执行

## ⚠️ 注意事项

1. **确保脚本可执行**
   ```bash
   chmod +x /root/.openclaw/workspace-researcher/auto-backup.sh
   ```

2. **检查日志文件权限**
   确保能写入 `/var/log/workspace-backup.log`

3. **网络连接**
   确保能访问GitHub，否则推送会失败

4. **定期检查**
   定期查看备份日志，确保备份正常运行
